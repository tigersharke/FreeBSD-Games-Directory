Known options:
--help             Display this help page and exit
--version          Display this program's version number and exit
--ms-per-frame=N   N milliseconds per animation frame [default: 55]
                   Min. 1, max. 1000.  50 ms means 20 frames per second
--full-screen      Attempt to use full screen mode [window mode]
--maze=M           Select maze M: 0 = original, 1 or 2 = alternatives
--rules=R          Rules to use (R must be 1 or 2, default is 1)
                   1 = original Afternoon Stalker rules
                   2 = 2008 updated Afternoon Stalker rules
--no-auto-pause    Do not pause automatically when window loses focus
